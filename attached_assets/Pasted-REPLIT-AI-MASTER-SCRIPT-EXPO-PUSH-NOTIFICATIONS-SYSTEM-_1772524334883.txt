1️⃣ DATABASE UPDATE

Add new column to users table:

ALTER TABLE users ADD COLUMN IF NOT EXISTS expo_push_token TEXT;

Also add to bookings table:

ALTER TABLE bookings ADD COLUMN IF NOT EXISTS reminder_sent BOOLEAN DEFAULT FALSE;
2️⃣ CREATE NOTIFICATION UTILITY

Create file:

/server/utils/expoPush.js

Inside it:

Use fetch or axios

Send POST request to:

https://exp.host/--/api/v2/push/send

Function must:

Accept:

title

body

tokens (array)

optional image

Send notifications in chunks (max 100 per request)

Handle errors safely

3️⃣ CREATE SAVE TOKEN ENDPOINT

Create route:

POST /notifications/register-token

Requires JWT.

Body:

{
  "expo_push_token": "ExponentPushToken[...]"
}

Server:

Validate token format

Save token to users.expo_push_token

7️⃣ ADMIN BROADCAST NOTIFICATIONS

Create admin-only endpoint:

POST /admin/send-notification

{
